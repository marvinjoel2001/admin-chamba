# Admin Web (Stitch -> React)

## Backend endpoints usados como referencia
- `GET /users`
- `GET /users/nearby/workers?lat=&lng=&radiusKm=`
- `GET /mobile/request-status`
- `GET /mobile/offers`
- `GET /mobile/tracking`
- `GET /mobile/worker/radar`
- `GET /notifications/status`

## Variables
- `VITE_API_URL=http://localhost:3000`
- `VITE_MAPBOX_TOKEN=...`

## Ejecutar
```bash
cd admin
npm install
npm run dev
```
